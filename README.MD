# Clock

This is a simple clock that displays the current time, created using HTML, CSS, and JavaScript.

## Features

- Displays the current time
- Customizable font and gradient background
- Fullscreen display with no borders

## Usage

1. Clone or download the repository
2. Open the `index.html` file in your browser to see the clock in action
3. To customize the clock, edit the `index.css` file to change the font and background gradient.

## Note

The clock updates every second, so the time will always be accurate.
